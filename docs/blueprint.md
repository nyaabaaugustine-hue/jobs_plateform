# **App Name**: ReactHire

## Core Features:

- Smart Job Search: Enable users to search jobs with real-time filtering based on keyword, location, job type, salary range, and more.
- AI Job Recommendations: Provide job seekers with AI-driven 'Jobs you might like' suggestions and skill gap insights, along with resume matching scores. Use a tool for deciding when to recommend.
- Job Seeker Profile Builder: Allow job seekers to build a comprehensive profile, including resume upload with PDF parsing, LinkedIn import, work experience timeline, education, skills tagging, and more.
- Application Tracker: Implement a mini ATS (Applicant Tracking System) for candidates to track their applications through different stages like Applied, Screening, Interview, Offer, Hired, and Rejected.
- Employer Job Posting: Enable employers to post jobs with a rich text editor, skill requirements, screening questions, salary range, location, and job visibility settings.
- Applicant Tracking System (ATS): Provide employers with an ATS to manage applicants, filter by skill match %, experience, and status, move candidates between stages, add internal notes, and rate candidates.
- Admin Job Moderation: Allow admin to review job posts and flag suspicious listings and use AI to auto-detect spam.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), reminiscent of technology and open communication.
- Background color: Light blue (#E0F7FA), provides a clean and professional backdrop.
- Accent color: Analogous cyan (#29E2D2), to highlight important actions and information.
- Body font: 'Inter', a sans-serif font providing a modern, neutral look suitable for long-form text.
- Headline font: 'Space Grotesk', a sans-serif font offering a techy and contemporary aesthetic. Should be used for headlines and shorter text segments.
- Crisp, modern icons representing job categories, industries, and user actions.
- Clean, grid-based layout optimized for readability and user experience, with clear visual hierarchy.
- Subtle transitions and loading animations to enhance user engagement.
